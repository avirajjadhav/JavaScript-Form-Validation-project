# JavaScript Form Validation

This is a simple form validation project built using HTML,CSS, and JavaScript.

## Features

* Name must contain at least 3 characters.
* Email must include a valid `@` symbol.
* Contact number must contain exactly 10 digits.
* Password must be at least 8 characters long.
* Confirm Password must match the password.

## Technologies Used

* HTML
* CSS
* JavaScript

This project helped me practice JavaScript conditions, `alert()` messages, and basic form validation.
